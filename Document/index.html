Install CoffeeCMS - top

Step 1:

Download CMS source code at https://github.com/coffeecms/cms , then unzip and upload all files & folders inside CoffeeCMS folder into your hosting root path

Step 2:

Go to http://yousite.com/install.php

Step 3:

At install step 1, type your database information into form then click 'Next Step' button

At install step 2, type your website information into form then click 'Apply' button

At install step 3, install completed, click 'Login to admin' button

Step 4:

Type username & password then login to Admin Panel

Step 5:

See your admin panel dashboard page

Go To Table of Contents

Install BulletinBoard - top

Step 1:

Login into your admin panel at http://yousite.com/admin

Step 2:

On left menu, go to Plugins -> click to All Plugins -> On list all plugins page, click to Activate BulletinBoard plugin

Step 3:

At install page, type your license key into License Key text box then click Next Step button

If you see message 'Create database complete...' then click Next Step button

If you see message 'Install complete...' then click Back to dashboard button

Go To Table of Contents
